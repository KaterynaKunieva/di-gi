# Di Gi – Landing Page

Responsive single-page landing website built using HTML, CSS, JavaScript/JQuery, and Owl Carousel.

Deployed to [GitHub Pages](https://katerynakunieva.github.io/di-gi).

The layout was developed based on the following [Figma design](https://www.figma.com/file/3jKk6CXwnDbcXng7GNrLt2/m-web-frontend-level1?type=design&node-id=0-625&mode=design).

## Technologies

- HTML5
- CSS3
- JavaScript (jQuery)
- Owl Carousel
